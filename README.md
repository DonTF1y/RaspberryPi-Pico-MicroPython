Repository containing examples and applications developed for the Raspberry Pi Pico using MicroPython as the programming language, Thonny as the development environment, and Wokwi for hardware simulation.

## Requirements

* Raspberry Pi Pico
* MicroPython firmware
* Thonny IDE
* Wokwi account (for simulation)

## Setup

1. Install the MicroPython firmware on the board.
2. Configure the MicroPython (Raspberry Pi Pico) interpreter in Thonny.
3. Run or simulate the `.py` files on real hardware or in Wokwi.

## Contents

* GPIO control
* Peripheral interfacing
* Basic embedded systems examples
